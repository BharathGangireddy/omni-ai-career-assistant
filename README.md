# OmniAI Career Assistant

Full-stack AI-powered resume builder and analyzer.

## 🚀 Features
- Resume Builder
- Resume Analyzer
- Job Matching Score
- Resume History

## 🧱 Tech Stack
- Frontend: Streamlit
- Backend: FastAPI
- Database: SQLite

## ▶️ Run

Backend:
uvicorn backend:app --reload

Frontend:
streamlit run app.py
